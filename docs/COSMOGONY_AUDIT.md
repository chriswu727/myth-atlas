# 创世时间线校勘记录

更新：2026-07-14

## 本轮范围

已从头到尾通读并复核八条核心时间线：华夏、希腊、北欧、两河、印度、埃及、日本、玛雅。其余十二条时间线保留在后续逐字校勘队列中。

本轮不把互相冲突的传统磨成一条顺滑故事。正文负责讲清叙事，`note` 负责标出文本年代、异本、晚出环节与编者拼接；跨文化对照页上的空白母题继续保留为空白。

## 已纠正的关键接缝

- 《古事记》的「独神成而隐身」只说神体隐去。高御产巢日神与神产巢日神后来都重新参与叙事，也都有后裔或子神；删去「不生子嗣，也不再管事」。
- 《波波尔·乌》的四位首祖没有父母，但并非被「念出来」；译文改回造物者以神力与灵质使其成形。
- 双子英雄活动在玉米人造成之前，不是「一个人也没有」的世界。四百少年升作星座，前文指为昴星团时原书自己紧跟一句保留语，现已补回。
- 《波波尔·乌》的初日面容炽热，热力不可承受，并非「照人不烫」；镜像说指今日余下的太阳不是真日。
- 两河洪水段明确分开《阿特拉哈西斯》的噪声、灾荒与芦苇墙，以及《吉尔伽美什》第十一板的六日七夜、尼穆什山与乌鸦。
- 北欧段补出两种杀狼法：《女占卜者的预言》是剑入狼心，《欺骗古鲁菲》是厚靴踏住下颚、扯裂巨口。
